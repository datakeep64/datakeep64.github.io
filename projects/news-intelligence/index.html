News Intelligence System

An end-to-end pipeline for processing news content — from raw article collection through to semantic search and retrieval. Integrates NLP and information retrieval techniques into a unified system.

Python BeautifulSoup spaCy NLTK Scikit-learn Sentence Transformers Pandas FAISS

Pipeline Architecture

Five sequential stages, each feeding into the next — designed so any stage can be swapped or extended independently.

01

Data Collection

Web scraping with BeautifulSoup targeting news sources. Structured extraction of headline, body, source, and publish date.

02

Text Preprocessing

Cleaning pipeline using NLTK and spaCy — tokenisation, stopword removal, lemmatisation, and noise filtering for consistent input to downstream models.

03

Topic Classification

Multi-class classifier trained on preprocessed article text. Labels articles by topic category using TF-IDF features and a Scikit-learn classifier.

04

Summarisation

Extractive summarisation to distill key sentences from each article, reducing length while preserving the core information.

05

Embedding-Based Search

Sentence Transformers encode articles into dense vectors. FAISS index enables semantic similarity search — retrieving relevant articles by meaning, not just keyword match.
