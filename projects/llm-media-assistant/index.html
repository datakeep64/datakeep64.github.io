LLM-Based Media Assistant

A system using the OpenAI API to assist with media content workflows — summarisation, groundedness verification, sentiment analysis, headline rewriting, and content restructuring. The verifier directly tackles the biggest blocker to newsroom AI adoption: catching hallucinated facts before they reach print.

Python OpenAI API Prompt Engineering Faithfulness / NLI NLP Streamlit

Capabilities

Seven distinct tasks, each shaping model output — or guarding it — for a specific media use case.

📄

Article Summarisation

Condenses long-form news articles into concise summaries at configurable length, streamed word-by-word as the model generates.

✅

Groundedness Verification

Fact-checks a summary against its source using atomic-claim decomposition and textual entailment (the FActScore / SummaC approach). Flags hallucinations down to the individual claim, cites a verbatim supporting span for each verdict, and scores overall groundedness 0–100.

🔍

Quote Integrity

Checks that every direct quotation survived the transform byte-for-byte — flagging Preserved / Altered / Fabricated quotes with a word-level diff. Deliberately deterministic (exact + fuzzy string matching, no LLM), since verbatim-ness is a string property, not a probabilistic judgment.

⚖️

Neutrality Analysis

Flags loaded language — editorializing verbs, emotive framing, intensifiers, weasel attribution, agency-hiding passive — against a curated lexicon, with an objectivity score 0–100 and an optional LLM rewrite that's re-scored to show the gain. Transparent by design: every deduction traces to a specific word.

📊

Sentiment Analysis

Chain-of-thought classification — the model cites evidence from the text before labelling tone, reducing inconsistent or hallucinated classifications.

✏️

Headline Rewriting

Generates alternative headline variants — neutral, clickbait, SEO-optimised — from the same article body.

🔄

Content Restructuring

Transforms article structure: converts inverted-pyramid format to narrative, or restructures for different audiences.
