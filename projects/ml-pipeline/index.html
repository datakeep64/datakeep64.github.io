ML Pipeline Automation System

A reproducible, production-style machine learning workflow — covering data preprocessing, model training, evaluation, and deployment — containerised with Docker and automated end-to-end via GitHub Actions CI/CD.

Python Docker GitHub Actions Scikit-learn FastAPI Pandas Pytest

Pipeline Stages

A linear workflow designed for reproducibility — every run is deterministic, containerised, and logged.

01

Data Preprocessing

Ingests raw datasets, handles missing values, encodes categoricals, scales numerical features. Outputs a versioned, cleaned dataset artifact.

02

Model Training

Trains a Scikit-learn classifier with configurable hyperparameters. Training script reads from the preprocessed artifact, logs parameters and run metadata.

03

Evaluation

Runs on held-out test split. Reports AUC, F1, precision, recall. Fails the CI run automatically if a threshold metric is not met.

04

Model Packaging

Serialises the trained model and wraps it in a FastAPI inference endpoint. Packaged into a Docker image ready for deployment.

05

CI/CD via GitHub Actions

On every push: installs dependencies, runs tests, executes the full pipeline, builds Docker image. Simulates a production deployment gate.
