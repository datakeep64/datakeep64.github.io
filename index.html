Projects

Applied Demonstrations

Each project here takes a concept from the IRS curriculum and builds it into something that can be explored hands-on. Code and implementation notes live on GitHub; interactive versions are hosted here.

Live
Genetic Algorithm — Travelling Salesman
Visualise how a population of routes evolves over generations toward an optimal path. Adjust mutation rate, population size, and watch convergence happen in real time.
→ Reasoning Systems · Evolutionary Computation
Live
Word Embedding Explorer
Compare how IMDB-trained, GPT-2, and Word2Vec embeddings position the same words in semantic space. Reduced to 2D with PCA and navigable interactively.
→ Cognitive Systems · Representation Learning
Live
N-Queens Constraint Solver
Step through how constraint propagation and backtracking work on a live chessboard. Adjustable board size, with each decision made visible.
→ Machine Reasoning · Constraint Satisfaction
Live
ASD Classification — DT vs kNN vs Bayes
Answer Q-Chat-10 screening questions and see how three different classifiers each arrive at a prediction — and why they sometimes disagree.
→ Machine Reasoning · Pattern Classification
Live
Knowledge Graph Explorer
Navigate entity relationships in a medical knowledge base. Drag nodes, search entities, and explore relationships — inspired by the PyKEEN/TransE assignment on UMLS.
→ Machine Reasoning · Knowledge Representation
In Progress
RAG Chatbot
A retrieval-augmented generation pipeline — upload a document, ask questions, and see which retrieved passages informed the answer.
→ Cognitive Systems · NLP & Retrieval
